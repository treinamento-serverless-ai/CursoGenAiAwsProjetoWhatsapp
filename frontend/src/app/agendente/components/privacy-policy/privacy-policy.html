🗓️ Agendente — Política de Privacidade

Última atualização: 04 de abril de 2026

1. Sobre o Projeto

O Agendente é um projeto de caráter pedagógico, desenvolvido para fins de treinamento e aprendizado de serviços serverless da AWS com IA Generativa. O código-fonte original é público e está disponível no repositório GitHub do projeto , distribuído sob a licença MIT. Qualquer pessoa pode utilizar, modificar e redistribuir o código livremente, sendo a responsabilidade pelo uso inteiramente de quem o utiliza.

2. Dados que Coletamos

Ao interagir com o Agendente via WhatsApp, podemos coletar e processar os seguintes dados:

  • Dados de identificação: nome e número de telefone do WhatsApp.
  • Dados de comunicação: mensagens de texto e áudio enviadas ao assistente via WhatsApp.
  • Dados de agendamento: informações sobre serviços solicitados, datas, horários e profissionais escolhidos.

3. Como Utilizamos os Dados

Os dados coletados são utilizados exclusivamente para:

  • Processar e responder às suas mensagens via WhatsApp.
  • Criar, consultar e cancelar agendamentos conforme sua solicitação.
  • Transcrever mensagens de áudio para interpretação pelo assistente de IA.

4. Compartilhamento de Dados

Seus dados podem ser processados por serviços de terceiros necessários para o funcionamento do aplicativo, incluindo a Meta Platforms (WhatsApp Business API) e a Amazon Web Services (serviços de nuvem e inteligência artificial). Não vendemos seus dados pessoais a terceiros.

5. Retenção de Dados

Por se tratar de um projeto pedagógico, os dados são retidos apenas durante o período de testes e aprendizado. Após a conclusão dos testes, o projeto pode ser encerrado e todos os dados armazenados serão removidos juntamente com a destruição da infraestrutura na AWS.

6. Exclusão de Dados

Você tem o direito de solicitar a exclusão dos seus dados pessoais a qualquer momento. Para isso, envie uma mensagem ao assistente no WhatsApp solicitando a exclusão dos seus dados.

7. Alterações nesta Política

Esta política pode ser atualizada periodicamente. A data da última atualização está indicada no topo desta página.
